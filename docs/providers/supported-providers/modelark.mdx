## Overview

ModelArk uses the BytePlus-hosted international ARK endpoint and shares the same API behavior as the Volcengine ARK provider in Bifrost.

### Supported Operations

| Operation | Non-Streaming | Streaming | Endpoint |
|-----------|---------------|-----------|----------|
| List Models | ✅ | - | `/models` |
| Text Completions | ✅ | ✅ | `/completions` |
| Chat Completions | ✅ | ✅ | `/chat/completions` |
| Responses API | ✅ | ✅ | Fallback to Chat Completions |
| Embeddings | ✅ | ❌ | `/embeddings` |
| Image Generation | ✅ | ❌ | `/images/generations` |
| Video Generation | ✅ | ❌ | `/contents/generations/tasks` |
| Video Retrieve / Download / Delete / List | ✅ | ❌ | `/contents/generations/tasks` |
| File Upload / List / Retrieve / Delete / Content | ✅ | ❌ | `/files` |
| Image Edit / Image Variation / Batch | ❌ | ❌ | - |

## Configuration

<Tabs>
<Tab title="Gateway">

```bash
curl --location 'http://localhost:8080/api/providers' \
--header 'Content-Type: application/json' \
--data '{
  "provider": "modelark",
  "network_config": {
    "base_url": "https://ark.ap-southeast.bytepluses.com/api/v3"
  },
  "keys": [
    {
      "name": "modelark-key-1",
      "value": "env.MODELARK_API_KEY",
      "models": [],
      "weight": 1.0
    }
  ]
}'
```

</Tab>
<Tab title="Go SDK">

```go
case schemas.ModelArk:
    return []schemas.Key{{
        Value:  *schemas.NewEnvVar("env.MODELARK_API_KEY"),
        Models: []string{},
        Weight: 1.0,
    }}, nil
```
